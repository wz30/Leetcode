## 1186. Maximum Subarray Sum with One Deletion

### Solution 1
- O(n)
```java
class Solution {
    public int maximumSum(int[] arr) {
        int max = getMax(arr);
        if(max <0) return max;
        //delete once
        int res = 0;
        int s0 = 0, s1 = 0;
        
        for(int a : arr){
            s1 = max(s0, s1+a);
            s0+=a;
            res = max(s0, res);
            res = max(s1, res);
            if(s0<0) s0=0;
            if(s1<0) s1 = 0;
        }
        
        return res;
    }
    int max(int a, int b){
        if(a>b) return a;
        else return b;
    }
    int getMax(int[] arr){
        int res = Integer.MIN_VALUE;
        for(int a: arr){
            if(a>=res) res = a;
        }
        return res;
    }
    //https://leetcode.com/problems/maximum-subarray-sum-with-one-deletion/discuss/377372/Java-AC-O(n)-time-and-O(1)-space-dp-solution
    //https://leetcode.com/problems/maximum-subarray-sum-with-one-deletion/discuss/377486/Java-DP-O(n)-Beat-100-detail-explanation
}
```

## 252. Meeting Rooms

### Solution 1
- using sort for two arrays
```java
class Solution {
    public boolean canAttendMeetings(int[][] intervals) {
        int n = intervals.length;
        int[] start = new int[n];
        int[] end = new int[n];
        for(int i = 0; i<n; i++){
            start[i] = intervals[i][0];
            end[i] = intervals[i][1];
        }
        Arrays.sort(start);
        Arrays.sort(end);
        /*
            0 6 11
            5 10 20
            make sure start[i] >= end[i-1]
        */
        for(int i = 1; i<n; i++){
            if(start[i] < end[i-1]) return false;
        }
        return true;
    }
}
```
